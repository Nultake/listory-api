# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Listory, please report it responsibly.

**Do NOT open a public issue.**

Instead, please email: [YOUR_EMAIL]

## Sensitive Data

This repository should never contain:
- API keys or secrets
- Database credentials
- `.env` files (only `.env.example` with empty values)
- Personal access tokens
- Private keys or certificates

If you find any sensitive data committed by mistake, please report it immediately.
